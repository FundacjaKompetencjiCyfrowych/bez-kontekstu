# Strona fundacji Bez Kontekstu

## Stack

- Turborepo (root)
- Next.js (apps/web)
- Sanity Studio (apps/studio)
- prettier (root)

### Next

Project bootstrapped with create-next-app

Root structure:

- Configuration files (ts, postcss, next, eslint)
- App folder (Next.js app router)
- Lib folder (utils, external libraries)
- Components folder (shared components, shadcn)

### Sanity

Project bootstrapped with create sanity

## Development

Run `pnpm dev` to start the development server

- Studio runs on port _3333_ by default
- Frontend app runs on port _3000_ by default

This project follows the Git Flow branching model
