# Podstrony aplikacji

## Struktura podstron:

1. `page.tsx` - Strona główna (Home)
2. `donors/page.tsx` - Dla Darczyńców
3. `manifest/page.tsx` - Manifest
4. `projekty/page.tsx` - Portfolio
5. `contact/page.tsx` - Kontakt
6. `cooperators/page.tsx` - Osoby współpracujące

## Zasady:

- Każda podstrona w osobnym folderze
- Używaj layout.tsx dla wspólnych elementów
- Dodawaj metadata dla SEO
- Testuj na urządzeniach mobilnych
